The Trend Analyzer Dashboard is a data analytics web app built using Python and the Streamlit library. It allows users to upload and analyze dataset values and visualize trends through interactive charts.
The app uses Pandas for data processing and provides features like line charts, moving averages, and statistical summaries (average, highest, lowest). It also generates simple insights based on the dataset.
This project helps in understanding data patterns and trends in an easy and visual way without complex coding.
