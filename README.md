Overview

The Trend Analyzer Dashboard is an interactive data analytics web application built using Python and the Streamlit library.
It allows users to upload datasets and visually analyze trends using dynamic charts and statistical summaries.

The project focuses on exploratory data analysis (EDA) and helps users understand patterns in numerical data in a simple and visual way.

Features

 Load dataset automatically (students.csv.csv)
 Select numeric columns dynamically
 Interactive trend line chart
 Moving average (data smoothing)
 Area chart visualization
 Statistical summary:
Average
Maximum
Minimum
 Auto-generated insights (data skew detection)
 Clean UI using custom CSS



Tech Stack

Programming Language: Python
Framework: Streamlit
Libraries:
Pandas (data processing & analysis)






How It Works

Dataset is loaded using Pandas
Numeric columns are automatically detected
User selects a metric
App generates:
Line chart (trend analysis)
Moving average chart
Statistical summary
System provides basic insights based on mean vs median
